# Hello World Demo

This repository contains a simple program that prints "Hello, World!" to the console.

## How to recreate this code

1. Make sure you have Python installed on your system.
2. Create a new file named `hello_world.py`.
3. Open the file in your favorite text editor and paste the following code:
   ```python
   print("Hello, World!")
   ```
4. Save the file.
5. Open your terminal or command prompt, navigate to the directory where you saved the file, and run:
   ```bash
   python hello_world.py
   ```
